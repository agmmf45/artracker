# دقيق (Daqeeq)

تطبيق ويب عربي (PWA) لتتبّع **العادات والمهام، المصاريف، التمارين، والسعرات** في مكان واحد — بإحساس gamified وخط **ثمانية**.

## الملفات
- **`index.html`** — التطبيق الكامل: ملف واحد مكتفٍ ذاتياً (HTML + CSS + JavaScript، وخط ثمانية مضمّن كـ base64). هذا هو كل ما تحتاجه لتشغيل التطبيق.
- `dashboard-design.html` / `workout-redesign.html` — نماذج تصميم (مرجعية فقط).

## التشغيل
افتح `index.html` في المتصفح. للتشغيل الكامل (تسجيل الدخول + حفظ البيانات + مساعد الذكاء الاصطناعي) يُخدَّم عبر **Cloudflare Worker** يوفّر واجهات `/api/*` (قاعدة بيانات D1) و proxy للـ AI.

## ملاحظات
- التطبيق mobile-first ويعمل كتطبيق ويب قابل للتثبيت (PWA).
- لا تضع مفاتيح API داخل `index.html` (تُكشف للعموم) — استخدم الـ Worker كوسيط.
